# Ecommerce Chatbot
A GenAI-powered customer service chatbot designed for an e-commerce clothing company. Built with **LangChain, Pinecone, Groq, llama3.3 70b model**. The chatbot provides product recommendations, processes orders, tracks shipments, and remembers past conversations for a seamless user experience.
